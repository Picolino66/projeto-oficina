    Componente Principal (AppComponent)

    O AppComponent é o componente raiz da aplicação Angular. Ele é responsável por controlar a navegação e o estado geral da aplicação, além de gerenciar temas (claro/escuro) e autenticação.

    Visão Geral

    O AppComponent contém a lógica para:

    • Alternar entre o modo claro e escuro.
    • Gerenciar o título da página com base na rota atual.
    • Navegar para a página de cadastro e logout.
    • Identificar se o usuário tem status de administrador.

    1. Estrutura do Componente

    Importações

    O componente importa vários módulos e serviços necessários para seu funcionamento, incluindo:

    • Router: Para gerenciar a navegação.
    • FuncionarioService: Para obter informações sobre funcionários.
    • LoginService: Para gerenciar o estado de autenticação.
    • Subscription: Para gerenciar assinaturas RxJS.

    Declaração do Componente

    @Component({
  selector: 'app-root',
  templateUrl: './app.component.html',
  styleUrls: ['./app.component.css']
})
export class AppComponent implements OnInit, OnDestroy {
  dark: boolean = true;
  private routeSub: Subscription | undefined;
  title = 'client-oficina-josue-1.0';
  adm = false;
  //...
}

    2. Funcionalidades Principais

    2.1 Alternância de Tema

    O componente oferece a funcionalidade de alternar entre modos de tema claro e escuro:

    toggleTheme() {
  document.body.classList.toggle('dark');
  this.dark = !this.dark;
}

    Essa função é chamada quando o usuário clica no botão de alternância de tema no template HTML.

    2.2 Gerenciamento de Navegação e Título da Página

    O AppComponent gerencia o título da página com base na rota atual usando o serviço Router:

    ngOnInit(): void {
  this.admin();
  this.routeSub = this.router.events.subscribe(event => {
    if (event instanceof NavigationEnd) {
      const pagina = this.router.url.slice(1).replace(/\//g, '->');
      if(pagina.includes("dashboard")){
        this.title = "Painel";
      } else if(pagina.includes("cliente")) {
        this.title = "Cliente";
      }
      //... outras condições
    }
  });
  document.body.classList.toggle('dark');
  initFlowbite();
}

    Esse método escuta eventos de navegação e atualiza o título da página de acordo com a rota ativa.

    2.3 Funções de Navegação

    O componente inclui métodos para navegar entre diferentes partes da aplicação:

    • cadastro(): Navega para a página de cadastro.
    • sair(): Realiza o logout do usuário e redireciona para a página inicial.
    cadastro() {
  this.router.navigate(["/cadastro"]);
}

sair() {
  this.login.logout();
  localStorage.clear();
  this.router.navigate(["/"]);
}
